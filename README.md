# 🍽️ مطعم ومطبخ العائلة - نظام الطلبات الإلكتروني

## 📁 هيكل المشروع

```
restaurant/
├── server.js              # الخادم الرئيسي (Node.js + Express)
├── package.json           # مكتبات المشروع
├── data/
│   └── db.json            # قاعدة البيانات (JSON)
├── public/
│   ├── index.html         # واجهة المينيو للزبائن
│   ├── admin.html         # لوحة التحكم (مخفية)
│   └── order-status.html  # صفحة تتبع الطلب
└── uploads/               # صور المنتجات (تنشأ تلقائياً)
```

---

## 🚀 تشغيل المشروع

### 1. تأكد من وجود Node.js على جهازك
```bash
node --version   # يجب أن يكون 16+
npm --version
```

### 2. تثبيت المكتبات
```bash
cd restaurant
npm install
```

### 3. تشغيل السيرفر
```bash
npm start
# أو للتطوير:
npm run dev
```

### 4. فتح الموقع
- **المينيو (للزبائن):** http://localhost:3000
- **لوحة التحكم:** http://localhost:3000/admin
- **طلب من طاولة:** http://localhost:3000/?table=t1

## 🌐 النشر على الإنترنت

### نشر على Railway (مجاني):
1. سجّل في https://railway.app
2. ارفع المشروع على GitHub
3. اربط المشروع بـ Railway
4. سيعمل تلقائياً!

### نشر على Render:
1. سجّل في https://render.com
2. "New Web Service" → ارفع المشروع
3. Build Command: `npm install`
4. Start Command: `npm start`

## ⚙️ الميزات الكاملة

### المينيو (واجهة الزبون):
- ✅ عرض المنتجات بتصميم فاخر
- ✅ فلترة حسب الفئة
- ✅ سلة تسوق كاملة
- ✅ طلب داخلي (QR) أو خارجي
- ✅ خيار توصيل (+6 شيكل)
- ✅ تتبع حالة الطلب

### لوحة التحكم:
- ✅ إحصائيات اليوم والشهر
- ✅ إدارة الطلبات (قبول/رفض/جاهز)
- ✅ إشعارات فورية + صوت
- ✅ إضافة/تعديل/حذف المنتجات
- ✅ نظام خصومات
- ✅ رفع الصور
- ✅ QR Code لكل طاولة
